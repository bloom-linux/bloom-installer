# bloom-installer

Installation and first-run experience for Bloom Linux.

Includes:
- **bloom-welcome** — Welcome app shown on first boot (keybinds, features, install tab)
- **bloom-install** — Graphical installer GUI (frontend for archinstall)
- **bloom-firstrun** — First-run setup wizard (runs after installation)

## Install

```
pacman -S bloom-installer
```

## Dependencies

`gtk4` `python` `python-gobject` `archinstall`
